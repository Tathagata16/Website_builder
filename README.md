# 🤖 AI Website Builder (MERN Stack)

An AI-powered website builder that generates fully functional websites from simple text prompts. Built using the MERN stack, this platform enables users to instantly create, preview, and deploy websites with a seamless experience.

---

## 🚀 Features

### 🤖 AI Website Generation

* Describe your idea in plain text
* AI generates a complete website instantly

### 💰 Credit-Based System

* Users spend credits to generate websites
* Scalable usage model for monetization

### 💳 Secure Payments Integration *(Coming Soon)*

* Purchase credits securely using Stripe Checkout

### 🚀 One-Click Deployment *(Coming Soon)*

* Instantly deploy generated websites

### 🎞 Premium Animations

* Smooth and modern UI powered by Motion animations

### 🔐 Authentication & Protected Routes

* Secure login/signup system
* Protected routes for authenticated users

### ☁️ Production Deployment

* Fully deployed frontend and backend on Render

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-website-builder.git
cd ai-website-builder
```

### 2. Install dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

---

### 3. Environment Variables

Create a `.env` file in the `server` folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
STRIPE_SECRET_KEY=your_stripe_key   # (for future implementation)
```

---

### 4. Run the application

#### Backend

```bash
cd server
npm run dev
```

#### Frontend

```bash
cd client
npm start
```
